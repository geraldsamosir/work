Selamat datang di Admin Panel BLOG ENGINE! Pada halaman ini, anda dapat melakukan berbagai operasi-operasi Manajamen Operasional

Adapun operasi operasi yang dapat dilakukan adalah sebagai berikut :

  • Tambah User Baru,
  • Kelola User berupa Penaikan Status User menjadi Admin (dan sebaliknya),
  • Kategori, yang meliputi :
    • Tambah Kategori Baru,
    • Edit Nama Kategori, dan
    • Hapus Kategori yang Tidak Terpakai.

Adapun keterbatasan operasi-operasi yang berkaitan dengan Edit Data User maupun Post tidak diberikan karena dianggap melanggar Privasi dari Pemilik Data atau Post yang bersangkutan.

Meskipun demikian, selaku Admin Anda tetap bertanggung jawab terhadap semua permasalahan yang terjadi dan berhubungan dengan Operasional Web.
